## Run Locally

Clone the project

```bash

  git clone https://github.com/myusuf4892/go-articles.git

```

Go to the project directory

```bash

  cd go-articles

```

Run application

```bash

  go run main.go

```

Go test application

```bash

chmod 777 ./test.sh

./test.sh

```
